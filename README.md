# Capacitação em JavaScript - FreeCodeCamp

Este repositório contém as soluções desenvolvidas durante o primeiro ciclo de capacitação em JavaScript, utilizando o curso **Algoritmos e Estruturas de Dados em JavaScript** da [FreeCodeCamp](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/).

## 📚 Conteúdo

Foram realizadas atividades envolvendo:

- ES6
- Basic Data Structures
- Basic Algorithm Scripting
- Object Oriented Programming
- Functional Programming
- Intermediate Algorithm Scripting

## 📂 Organização do Repositório

Cada pasta contém o código utilizado para resolver a atividade correspondente.

- `n1/questao1`
- `n2/questao2`
- ...

## 🖼️ Assets

Na pasta `assets` estão disponíveis:

- Certificado de conclusão do curso
